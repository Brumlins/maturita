Skripty v Linuxu

// Klíčové pojmy

  • Bash skript: shebang, proměnné, $1..$N
  • if/elif/else, for, while – podmínky a cykly
  • grep, sed, awk, sort, cut – filtrace textu
  • Roury (|), přesměrování (>, >>, 2>)

Proč skripty?

Automatizace opakujících se úloh, snížení chybovosti, správa serverů a stanic. Shell skript = soubor bash příkazů vykonaných postupně.

Základy bash skriptu

#!/bin/bash
# Shebang – říká shellu, který interpret použít

# Proměnné (bez mezer kolem =!)
jmeno="Jan"
vek=25
echo "Jmenuji se $jmeno a je mi $vek let."

# Vstup od uživatele
read -p "Zadej jméno: " jmeno

# Poziční parametry
echo "Skript: $0  Arg1: $1  Počet: $#"
echo "Exit kód: $?"

Podmínky

if [ "$cislo" -gt 100 ]; then
    echo "Velké"
elif [ "$cislo" -gt 50 ]; then
    echo "Středně velké"
else
    echo "Malé"
fi

# Testovací operátory
[ -f soubor ]     # je to soubor?
[ -d adresář ]    # je to adresář?
[ -z "$str" ]     # prázdný řetězec?
[ $a -eq $b ]     # čísla se rovnají?
[ "$s" = "abc" ]  # řetězce se rovnají?

Cykly

# For – seznam / rozsah
for i in {1..10}; do echo $i; done
for soubor in /tmp/*.log; do rm "$soubor"; done

# While
while [ "$odpoved" != "ano" ]; do
    read -p "Pokračovat? " odpoved
done

Filtrace textu

NástrojPoužitíPříklad
grepHledání vzorugrep -r "error" /var/log/
sedNahrazování, mazání řádkůsed 's/foo/bar/g' file
awkZpracování sloupcůawk '{print $1,$3}' file
sortŘazenísort -n -r file
cutVýřez sloupcůcut -d: -f1,7 /etc/passwd
wcPočítání řádků/slovwc -l soubor

Roury a přesměrování

příkaz > soubor.txt       # stdout → soubor (přepis)
příkaz >> soubor.txt      # stdout → soubor (přidání)
příkaz 2> chyby.txt       # stderr → soubor
příkaz &> vše.txt         # stdout + stderr

# Pipeline
cat /etc/passwd | grep bash | cut -d: -f1 | sort
ps aux | grep nginx | awk '{print $2}' | xargs kill
