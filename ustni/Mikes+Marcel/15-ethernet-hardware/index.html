Ethernet – síťový hardware

📌 Klíčové body

  • Hub (L1) vs Switch (L2) vs Router (L3)
  • CSMA/CD – při full-duplex switchi odpadá
  • STP/RSTP – zabraňuje smyčkám, blokuje porty
  • LACP – agregace linek, VLAN 802.1Q – trunk/access

Síťová zařízení L1 a L2

Hub (L1)

Rozbočovač – přijme bit a pošle ho na všechny porty. Jedna kolizní doména = poloduplexní. Dnes se nepoužívá.

Switch (L2)

Přepínač – učí se MAC adresy, posílá rámec jen na správný port. Každý port = vlastní kolizní doména. Fullduplexní.

Router (L3)

Směrovač – propojuje sítě (LAN/WAN), pracuje s IP adresami. NAT, DHCP, firewall, směrování.

CSMA/CD – kolizní doména

CSMA/CD = Carrier Sense Multiple Access / Collision Detection
1. Poslouchej médium (CS) → 2. Je-li volné, vysilej → 3. Detekuj kolizi (CD) → 4. Jam signal → 5. Random backoff a zkus znovu.
Dnes při full-duplex Ethernetu (switch ↔ NIC) kolize nevznikají.

MAC tabulka switche

# Cisco IOS – zobrazení MAC tabulky
show mac address-table
show mac address-table dynamic

# Linux bridge
bridge fdb show

STP – Spanning Tree Protocol

STP (802.1D) zabraňuje smyčkám v L2 síti. Volí Root Bridge (nejnižší Bridge ID), blokuje redundantní porty. Modernější verze: RSTP (802.1w) – rychlejší konvergence, MSTP (802.1s) – per-VLAN instance. 
# Cisco IOS
show spanning-tree
show spanning-tree vlan 10

LACP – agregace linek

LACP / 802.3ad = Link Aggregation Control Protocol. Sdružuje více fyzických linek do jednoho logického kanálu (Port-Channel / Bond). Výhody: větší přenosová kapacita + redundance. 
# Linux – bonding/teaming
ip link add bond0 type bond
ip link set bond0 type bond mode active-backup
ip link set eth0 master bond0
ip link set eth1 master bond0
ip link set bond0 up

# Stav
cat /proc/net/bonding/bond0

VLAN – virtuální LAN

VLAN = logické oddělení sítě na L2 úrovni. Rámce jsou tagované 802.1Q (4 byte tag s VLAN ID 1–4094). Trunk port = nese více VLAN, Access port = jedna VLAN. 
# Cisco IOS VLAN konfigurace
vlan 10
 name SERVERS
interface Fa0/1
 switchport mode access
 switchport access vlan 10

interface Gi0/1
 switchport mode trunk
 switchport trunk allowed vlan 10,20,30
