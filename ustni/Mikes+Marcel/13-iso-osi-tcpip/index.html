ISO/OSI a TCP/IP modely

📌 Klíčové body

  • 7 vrstev OSI: fyzická → linková → síťová → transportní → relační → prezentační → aplikační
  • 4 vrstvy TCP/IP: přístup → internet → transport → aplikace
  • Enkapsulace: data → segment → paket → rámec → bity
  • OSI = teoretický referenční model, TCP/IP = praxe

Enkapsulace dat

Při odesílání data procházejí vrstvami shora dolů – každá vrstva přidá svoji hlavičku (encapsulation). Na druhé straně se hlavičky odstraňují zdola nahoru (decapsulation).

Transport:  [TCP hlavička][DATA]              ← Segment
Síťová:     [IP hlavička][TCP hlavička][DATA] ← Paket
Linková:    [Eth hlavička][IP][TCP][DATA][FCS] ← Rámec
Fyzická:    101011001010110... (bity na kabelu)

OSI vs TCP/IP – proč to vědět?

OSI = referenční model (teoria), 7 vrstev – detailně popisuje funkce. Nikdy nebyl plně implementován jako celek, ale používáme ho pro popis a troubleshooting.
TCP/IP = praktický model, 4 vrstvy – skutečně implementovaný základ internetu. Vznikl dříve než OSI, proto ne vždy přesně odpovídá.
