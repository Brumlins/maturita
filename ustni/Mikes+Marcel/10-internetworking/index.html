Internetworking

// Klíčové pojmy

  • IP = základ internetworkingu
  • IPv4 privátní: 10/8, 172.16/12, 192.168/16
  • IPv6: 128 bitů, bez NAT, SLAAC
  • Směrování: statické vs RIP/OSPF/BGP

Co je internetworking?

Propojení více různých sítí (LAN, WAN) s různými protokoly a infrastrukturami. Internet je největším příkladem. Základ: protokol IP zajišťuje jednotný rámec.

Zvláštní: 127.0.0.1 = loopback, 169.254.x.x = APIPA (bez DHCP), 255.255.255.255 = broadcast

IPv6

Proč IPv6?

  • IPv4 vyčerpány 2011 (~4.3 mld. adres)
  • IPv6 = 128 bitů = 3.4 × 10³⁸ adres
  • Zápis: 8 skupin po 4 hex číslic (2001:db8::1)

Výhody IPv6

  • NAT nepotřeba – každé zařízení má veřejnou IP
  • Auto-konfigurace (SLAAC)
  • Lepší mobilita (stálá IP při cestování)
  • IPsec vestavěný

NAT – Network Address Translation

Router překládá privátní IP:port ↔ veřejná IP:port. Umožňuje sdílet jednu veřejnou IP pro celou LAN.
SNAT = masquerade (odchozí), DNAT = port forwarding (příchozí)
