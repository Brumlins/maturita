Polovodičové paměti

// Klíčové pojmy

  • Volatilní (RAM) vs nevolatilní (ROM, Flash)
  • ROM → PROM → EPROM → EEPROM → Flash
  • SRAM (cache L1/L2/L3) vs DRAM (operační RAM)
  • DDR1 → DDR2 → DDR3 → DDR4 → DDR5

Základní dělení pamětí

Podle volatility

  • Volatilní – při odpojení napájení ztrácí data (RAM)
  • Nevolatilní – data drží bez proudu (ROM, Flash, EEPROM)

Podle přístupu

  • RAM – Random Access Memory – přímý přístup kdekoliv okamžitě
  • SAM – Sequential Access – sekvenčně (pásky, dnes vzácné)

Paměti ROM – nevolatilní

TypPopis
ROMData vložena při výrobě, nelze měnit. BIOS starý.
PROMProgramovatelná uživatelem, ale pouze jednou (přepálení propojek).
EPROMMazatelná UV zářením (okénko v čipu), znovu programovatelná.
EEPROMElektricky mazatelná po bajtech – konfigurace zařízení.
FlashModerní EEPROM – maže se po blocích, levná, rychlá. SSD, USB disky.

RychlostExtrémně vysokáNižší

Vývoj DRAM

SDRAM
Synchronní s hodinami procesoru.
DDR
Double Data Rate – přenos na náběžné i sestupné hraně hodinového signálu (2× rychlejší než SDRAM).
DDR2 → DDR3 → DDR4 → DDR5
Každá generace = vyšší rychlost, nižší napájení, vyšší kapacita.
