Služby, CRON a správa balíčků

📌 Klíčové body

  • systemctl start/stop/enable/disable/status
  • Unit soubory: [Unit], [Service], [Install]
  • journalctl – logy systémových služeb
  • Crontab: minuta hodina den měsíc den_týdne

Systemd – správa služeb

systemd = init systém a správce služeb moderního Linuxu (PID 1). Nahradil SysV init. Paralelní start služeb, závislosti, journaling. 
# Základní ovládání
systemctl start   nginx
systemctl stop    nginx
systemctl restart nginx
systemctl reload  nginx      # reload konfigurace bez restartu

# Povolení při startu
systemctl enable  nginx      # autostart
systemctl disable nginx
systemctl mask    nginx      # zablokuj (nelze spustit)

# Stav a info
systemctl status nginx
systemctl is-active nginx
systemctl is-enabled nginx

# Výpis
systemctl list-units --type=service --state=running
systemctl list-unit-files --type=service

Unit soubory

# /etc/systemd/system/mojesluzba.service
[Unit]
Description=Moje služba
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/app
ExecStart=/opt/app/server
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
# Po vytvoření/úpravě unit souboru
systemctl daemon-reload
systemctl enable --now mojesluzba

Logy – journald

journalctl -u nginx              # logy konkrétní služby
journalctl -f                    # follow (live)
journalctl -n 50                 # posledních 50 řádků
journalctl --since "1 hour ago"
journalctl -p err                # jen chyby

CRON – plánování úloh

Crontab syntaxe: minuta hodina den měsíc den_týdne příkaz 
# Editace crontabu
crontab -e          # edituj crontab aktuálního uživatele
crontab -l          # zobraz crontab
crontab -u jan -e   # crontab uživatele jan (root)

# Příklady
* * * * *   /skript.sh               # každou minutu
0 * * * *   /skript.sh               # každou hodinu
0 2 * * *   /backup.sh               # každý den ve 2:00
0 2 * * 0   /weekly.sh               # každou neděli ve 2:00
*/5 * * * * /check.sh                # každých 5 minut
0 2 1 * *   /monthly.sh              # 1. v měsíci ve 2:00

Správa balíčků – apt (Debian/Ubuntu)

apt update                     # aktualizuj seznam balíčků
apt upgrade                    # aktualizuj nainstalované
apt install nginx              # instalace
apt remove nginx               # odinstalace (zachová konfigurace)
apt purge nginx                # odinstalace + konfigurace
apt autoremove                 # odstraň nepotřebné závislosti
apt search php                 # hledání balíčků
apt show nginx                 # info o balíčku
dpkg -l                        # seznam nainstalovaných
