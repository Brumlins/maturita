Dávkové zpracování (RJE)

Data jsou sbírána, zpracována jako celek (dávka) bez uživatelského zásahu. Použití: fakturace, finanční výpočty, generování reportů. Výhoda: efektivita CPU, bez nutnosti interakce.

Klient / Server

Klient

Iniciuje požadavek, zobrazuje výsledek. Prohlížeč, e-mailový klient, databázová aplikace.

Server

Čeká na požadavky, zpracovává, vrací odpověď. Apache, MySQL, SMTP server. ✗ SPOF.

Peer-to-Peer (P2P)

Každý uzel = klient i server. Žádný centrální server. Odolné, škáluje samo. BitTorrent, blockchain, Napster, I2P.
