Zobrazovací jednotky

// Klíčové pojmy

  • Rozlišení, obnovovací frekvence, pozorovací úhel, odezva
  • CRT: elektronové dělo + luminofor (historické)
  • LCD: TN (rychlé) vs IPS (barvy/úhly); podsvícení
  • OLED: pixel svítí sám, absolutní černá, riziko burn-in

CRT (Cathode Ray Tube)

Elektronové dělo vystřeluje svazek elektronů → elektromagnety ho usměrňují → dopadá na luminofor = bod zazáří.
✓ Skvělý kontrast, téměř nulová odezva. ✗ Obrovské rozměry, velká spotřeba, rentgenové záření. Dnes nevyrábí.

LCD (Liquid Crystal Display)

1. Zdroj světla (podsvícení) → 2. Polarizační filtr → 3. Tekuté krystaly (natáčí se elektrickým polem → mění polarizaci) → 4. Druhý filtr + RGB subpixely
Typ LCDVýhodyNevýhody
TNLevné, rychlá odezva (vhodné pro hráče)Špatné pozorovací úhly a barvy
IPSSkvělé barvy, pozorovací úhly ~180°Pomalejší, dražší
VAVysoký kontrastGhosting, pomalejší

OLED (Organic Light Emitting Diode)

Každý pixel svítí sám (organické sloučeniny vyzařují světlo při průtoku proudu).
✓ Absolutní černá (pixel se vypne), nekonečný kontrast, tenké a ohebné panely.
✗ Vypalování obrazu (statické prvky), modrá složka stárne nejrychleji.

E-ink a grafická karta

E-ink

Mikrokapsle s černým a bílým pigmentem. Energii spotřebuje jen při překreslení. Výborný na slunci, neoptiká oči. Kindle, čtečky.

GPU

Integrovaná (součást CPU, sdílí RAM) – kancelář. Dedikovaná (NVIDIA/AMD, vlastní VRAM) – hry, 3D, střih.
