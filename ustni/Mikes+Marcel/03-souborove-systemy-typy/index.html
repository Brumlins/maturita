Typy souborových systémů

📌 Klíčové body

  • FAT32, exFAT, NTFS, ext4, XFS, Btrfs, ZFS, APFS
  • Žurnálování – ochrana konzistence dat
  • Copy-on-Write (CoW) – snapshoty, atomický zápis
  • Praktické příkazy: mkfs, fsck, df, mount

Co je souborový systém?

Souborový systém = způsob organizace a ukládání dat na úložišti (HDD, SSD, USB). Určuje, jak se soubory pojmenovávají, ukládají, vyhledávají a zabezpečují.

Žurnálování (Journaling)

Žurnál = log operací, které se chystají provést. Při pádu systému lze snadno obnovit konzistentní stav bez kontroly celého disku (fsck).
Typy: Write-back (jen metadata) → Ordered → Full journal (metadata + data).

Copy-on-Write (CoW)

CoW: při zápisu se nevymaže původní blok – zapíše se nový blok, poté se aktualizuje ukazatel. Původní data zůstávají (snapshoty, atomický zápis). Btrfs, ZFS, APFS.

Praktické příkazy pro FS

# Zobrazit připojené FS
df -hT

# Informace o souborovém systému
tune2fs -l /dev/sda1        # ext4
xfs_info /dev/sda2          # XFS

# Formátovat oddíl
mkfs.ext4 /dev/sdb1
mkfs.xfs  /dev/sdb2

# Kontrola a oprava (ext4)
fsck.ext4 /dev/sda1

# Připojit / odpojit
mount /dev/sdb1 /mnt/data
umount /mnt/data
