Správa procesů – příkazy

📌 Klíčové body

  • ps aux, top, htop – zobrazení procesů
  • Signály: SIGTERM (15), SIGKILL (9), SIGHUP (1)
  • kill, killall, pkill – ukončení procesů
  • fg, bg, jobs – správa na pozadí

Zobrazení procesů

# Snapshot procesů
ps aux                      # všechny procesy, detailní
ps -ef                      # formát se PPID
ps -eo pid,ppid,ni,stat,cmd # vlastní sloupce

# Interaktivní monitor
top                         # tradiční
htop                        # barevný, myš (apt install htop)

# Strom procesů
pstree -p                   # zobrazí PID

# Procesy konkrétního uživatele
ps -u jan

Signály a kill

SignálČísloVýznam
SIGHUP1Znovu načti konfiguraci (hangup)
SIGINT2Přerušení (Ctrl+C)
SIGQUIT3Ukončení s core dump (Ctrl+\)
SIGKILL9Okamžité zabití – nelze zachytit!
SIGTERM15Žádost o ukončení (výchozí pro kill)
SIGSTOP19Pozastavení – nelze zachytit
SIGCONT18Pokračování (fg/bg)
# Poslat signál procesu
kill 1234                   # SIGTERM (15) – slušné ukončení
kill -9 1234                # SIGKILL – násilné zabití
kill -HUP 1234              # reload konfigurace
killall firefox             # kill podle jména
pkill -u jan                # kill všech procesů uživatele jan

Popředí a pozadí (fg/bg)

# Spustit na pozadí
./skript.sh &

# Přesunout do pozadí (suspenduj Ctrl+Z, pak bg)
Ctrl+Z          # pozastaví aktuální proces
bg              # pustí ho na pozadí
fg              # přenese zpět do popředí
fg %2           # přenese job číslo 2

# Seznam jobů
jobs -l

Sledování zdrojů

# CPU a paměť
vmstat 2 5          # každé 2s, 5 vzorků
free -h             # přehled RAM
cat /proc/meminfo   # detaily paměti
lscpu               # info o CPU

# Disk I/O
iostat -x 2
iotop               # jako top, ale pro I/O

# Síťové připojení
ss -tulnp           # otevřené porty a procesy
netstat -tulnp      # starší alternativa
