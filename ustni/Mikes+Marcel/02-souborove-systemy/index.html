Co je souborový systém?

Souborový systém (FS) = způsob organizace a ukládání dat na úložišti. Určuje pojmenování, ukládání a zabezpečení souborů.

Žurnálování

Žurnál = log operací před jejich provedením. Při pádu systému lze obnovit konzistentní stav bez kontroly celého disku. Typy: Write-back (jen metadata) → Ordered → Full (metadata + data).

Copy-on-Write (CoW)

Při zápisu se nevymaže původní blok – zapíše se nový, pak se aktualizuje ukazatel. Původní data zůstávají → atomické snapshoty. Používá: Btrfs, ZFS, APFS.

Inode

Inode = datová struktura pro každý soubor. Ukládá: UID, GID, oprávnění, velikost, časy (atime/mtime/ctime), ukazatele na datové bloky.
Nezahrnuje jméno souboru (to je v adresáři). Hard link = dvě jména → jeden inode.

Praktické příkazy

df -hT                    # zobraz FS a jejich obsazenost
lsblk                     # seznam blokových zařízení
mkfs.ext4 /dev/sdb1       # formátovat oddíl
mount /dev/sdb1 /mnt/     # připojit
stat soubor.txt           # info o inode souboru
tune2fs -l /dev/sda1      # info o ext4 FS
