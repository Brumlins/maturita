// Klíčové pojmy

  • DHCP DORA: Discover→Offer→Request→Ack
  • ARP: IP→MAC (broadcast req, unicast reply)
  • DNS záznamy: A, AAAA, MX, CNAME, PTR

DHCP – přidělení IP adresy

DHCP automaticky přiděluje IP, masku, bránu a DNS. UDP port 67 (server), 68 (klient).

DORA – 4 kroky

KrokZprávaKdoPopis
1DiscoverKlient → BroadcastHledám DHCP server (src: 0.0.0.0)
2OfferServer → KlientNabízím IP adresu a konfiguraci
3RequestKlient → BroadcastPřijímám nabídku
4AcknowledgeServer → KlientPotvrzuji, platí lease time

ARP – překlad IP → MAC

ARP zjistí MAC adresu pro danou IP v rámci LAN. Data se na síti posílají podle MAC adres! 
ARP Request  (broadcast FF:FF:FF:FF:FF:FF):
  "Kdo má IP 192.168.1.10? Řekni AA:BB:CC:DD:EE:FF"
ARP Reply    (unicast):
  "Já mám tu IP, moje MAC je 11:22:33:44:55:66"

ip neigh show            # ARP cache (Linux)
