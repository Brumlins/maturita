Správa uživatelů

// Klíčové pojmy

  • root (UID 0), systémoví, běžní uživatelé
  • /etc/passwd, /etc/shadow, /etc/group
  • useradd, usermod, userdel, passwd, groupadd
  • chmod, chown, SUID/SGID/Sticky, ACL (setfacl)

Typy uživatelů v Linuxu

root (UID 0)

Superuživatel – neomezená práva. sudo -i nebo su -

Systémoví (UID 1–999)

Spouštějí služby (www-data, mysql). Bez domovského adresáře a shellu.

Běžní uživatelé (1000+)

Omezená práva – jen ke svým souborům a nastavením.

Klíčové soubory

SouborObsah (formát)
/etc/passwdjméno:x:UID:GID:GECOS:home:shell
/etc/shadowhashovaná hesla + expirace (jen root čte)
/etc/groupjméno:x:GID:členové
/etc/sudoerskdo smí volat sudo a pro jaké příkazy

Správa uživatelů – příkazy

# Vytvoření / mazání
useradd -m -s /bin/bash -G sudo jan    # -m = domovský adresář
passwd jan                              # nastavit heslo
userdel -r jan                          # -r = smaž i domov

# Úprava
usermod -aG docker jan      # přidat do skupiny
usermod -L jan              # zamknout účet
usermod -U jan              # odemknout

# Skupiny
groupadd vyvojari
gpasswd -a jan vyvojari     # přidat uživatele do skupiny
gpasswd -d jan vyvojari     # odebrat

# Přepnutí
su - jan                    # přepnout na jana
sudo -u jan příkaz          # spustit jako jan

Oprávnění (chmod/chown)

ls -l → -rwxr-xr-- jan users 1234 soubor.txt
r=4 čtení, w=2 zápis, x=1 spuštění
Skupiny: vlastník | skupina | ostatní 
chmod 755 skript.sh         # rwxr-xr-x
chmod 644 soubor.txt        # rw-r--r--
chmod u+x,go-w soubor.txt  # symbolicky
chown jan:users soubor.txt  # změna vlastníka
