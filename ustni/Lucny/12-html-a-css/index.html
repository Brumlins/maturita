HTML a kaskádové styly

12.1 Jazyk HTML, jeho verze, syntax a validita

HTML (Hyper Text Markup Language) se používá pro tvorbu webových stránek. Termín hypertext odkazuje na schopnost odkázat na jiné texty a stránky. Jedná se o značkovací jazyk – dokument je strukturován pomocí značek (tagů).

První verzi publikoval Tim Berners-Lee v roce 1991 (HTML 0.9) – možnost vkládat obrázky, odkazy, tvořit logické úrovně. XHTML je založena na jazyce XML (definice: strict, transitional, frameset). Dnes se používá verze HTML 5.

Tagy

Skládají se ze „zobáčků" a názvu, např. <p></p>. Mohou být párové a nepárové. Určují význam textu, který obklopují. Cokoliv není ohraničeno značkou, bude zobrazeno na stránce.

  • Strukturální – definují strukturu: <p>, <h1>.
  • Sémantické – definují povahu prvku: <title>, <article>.
  • Stylistické – mění vzhled; dnes spíše CSS; nekompatibilní s alternativními prohlížeči: <b>.
  • Atribut – přiřazuje vlastnosti elementu; píše se do počátečního tagu; používají se uvozovky (např. class).
  • Element – celá sekvence tagu a atributů. Typy: blokové (zalomí text: <h1>, <div>, <ol>), inline (zvýraznění uvnitř textu: <i>, <em>, <b>), nahrazované (nahrazeny obsahem, důležité rozměry: <img>).
  • Entity – vyjadřují pomocí sekvence znaků jeden symbol: &copy;

Validita – ověřujeme pomocí validátoru (speciální program, např. W3C) – validace pomocí URL odkazu nebo souboru. Výsledkem je výpis přesně popsaných závad. Nevalidní kód může ohrozit zážitek zrakově postižených a sabotovat vyhledávání.

12.2 Struktura webové stránky, HTML elementy

  • Deklarace – sděluje prohlížeči, že otevřel HTML dokument; rozlišuje verzi: <!DOCTYPE html>
  • Kořenový element – reprezentuje celý dokument: <html></html>
  • Hlavička – obsahuje metadata vztahující se k celému dokumentu (název, linky, klíčová slova, kaskádové styly). Meta tagy jsou nepárové značky: starají se o správné zobrazení, indexování, informace pro vyhledávače (keywords), nastavují kódování (UTF-8), přizpůsobují zobrazení.
  • Tělo dokumentu – obsahuje finální obsah dokumentu. Komentáře: <!-- komentář -->

Blokové HTML elementy

  • Nadpis – šest úrovní; hlavní se používá pouze 1×; pomáhá vyhledávání.
  • Odstavec, oddíl – odstavce slouží k členění delšího textu; oddíl poté celé stránky (CSS styly). V HTML 5 se používá mnoho alternativ: <article>, <aside>.
  • Seznam – uspořádaný <ol>, neuspořádaný <ul>, popisný <dl>, <dt>, <dd>.
  • Tabulka – základní element <table>; úrovně <thead>, <tbody>, <tfoot>; řádky <tr>; buňky <td>; colspan pro určení rozpětí.

Inline elementy se vkládají do blokových – ohraničují kratší úsek textu (<span>). Zajímavé elementy: <strong>, <cite> (název citovaného díla), <code> (programový kód). Sémantické značky: <details>, <article>, <nav>.

12.3 Odkazy, navigace

K tvorbě odkazů se používá značka <a>; povinný atribut href definuje URL adresu; atribut target určí, zdali se odkaz otevře v novém okně (_blank = ano).

  • Absolutní – obsahují úplnou URL adresu; odkazují na dokumenty na jiném serveru.
  • Relativní – na stejném serveru; nekompletní URL, relativní vzhledem k výchozí stránce.
  • Na záložky – vyznačené místo ve stejném dokumentu; určité sekce webu.

Typy navigace

  • Hlavní – u větších webů může obsahovat menu; u největších se dělí na globální a lokální.
  • Hierarchická – větvení; uživatel přechází na specifičtější kategorie (Alza).
  • Kontextová – seznam souvisejících odkazů; tvorba zkratek; nezávislá na hlavní navigaci.
  • Drobečková – ukazuje pozici na webu: Domů > PC > herní > klávesnice.

12.4 Kaskádové styly (CSS), integrace do HTML, syntax, selektory

CSS umožňují formátování stránek psaných v jazyce HTML, XHTML nebo XML; navrženy W3C. Pointou je oddělení vzhledu od struktury stránky. Hlavní výhody: rozsáhlejší formátovací možnosti, dynamičnost, lepší kompatibilita.

Integrace CSS do HTML

  • Vložené styly – atribut style; platí pro jednu značku.
  • Styly na úrovni dokumentu – píší se do hlavičky; lze je překrýt vloženým stylem.
  • Externí styly – vložení souboru/odkazu; aplikovatelné na celý projekt; příkaz link nebo @import v tagu style.

Syntax CSS

Pravidla se skládají ze selektoru a deklarací; deklarace se skládají z vlastnosti a hodnoty (vlastnost:hodnota). Platí ta deklarace, která se zpracuje poslední (spodní). Příkaz !important zajistí nepřepsatelnost hodnoty.

  • Třídy – .class pro více elementů; #id pouze pro jeden.
  • Pseudotřídy – speciální stav nebo část elementu: :hover, :first-line.
  • Dědičnost – třída dědí vlastnosti elementu, ve kterém je vložena.
  • Selektor – slouží k výběru prvku, na který chceme aplikovat formátování. Kombinátory: > (pro všechna p vnořená v div), + (pro p přímo po div), ~ (všechna p příbuzná s div).

Úprava stránky pomocí CSS

  • Vlastnosti písma (font-family), textu (text-align), blokových elementů (margin).
  • Délkové jednotky: %, pt, px.
  • Barvy: slovně (black, red), #rgb (0 až f), rgb (0 až 255).
  • URL: absolutní, relativní vůči serveru nebo aktuálnímu adresáři.

12.5 Layout webové stránky, zásady webdesignu

Layout označuje rozmístění prvků na stránce. Dobrý layout kombinuje vzhled, použitelnost a pravidla informační architektury.

  • Dříve: Frames (rozdělení prohlížeče na nezávislé dokumenty) a tabulkový layout (<table>).
  • Moderní tvorba: schématické značky (header, main, section) + CSS.
  • CSS Flexbox – ideální pro jednorozměrné rozložení (řádek nebo sloupec); zarovnávání v navigaci, fotogalerie, centrování prvků.
  • CSS Grid – pokročilý dvourozměrný layout systém; složitá mřížka s řádky a sloupci pro celou strukturu stránky.
  • Responzivní design – layout se plynule přizpůsobuje zařízením; využívá Media Queries (mění CSS pravidla dle aktuální šířky displeje).

Zásady dobrého webdesignu

  • Přehlednost a srozumitelnost, ovladatelnost, dostupnost.
  • Rychlost, typografie a konzistence.
