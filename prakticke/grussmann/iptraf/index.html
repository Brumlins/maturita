iptraf — síťový provoz

🐧 Linux 📊 monitoring sítě

Instalace

Instalace iptraf-ng 
# iptraf-ng je aktualizovaná verze původního iptraf:
$ apt-get install iptraf-ng

# Starší verze (Debian/Ubuntu):
$ apt-get install iptraf

Spuštění

Spuštění iptraf 
# Musí běžet jako root:
$ sudo iptraf-ng

# Monitorovat konkrétní rozhraní přímo:
$ sudo iptraf-ng -i eth0

# Všechna rozhraní najednou:
$ sudo iptraf-ng -i all

Filtry a konfigurace

Filtrování podle portu / IP 
# Sledovat pouze HTTP provoz (port 80):
$ sudo iptraf-ng -f -i eth0
# Filtry se konfigurují v menu: Filters → IP

# Logovat výstup do souboru:
$ sudo iptraf-ng -L /var/log/iptraf.log -i eth0
