Apache2

🐧 Linux 🌐 webový server

Instalace Apache2

Instalace + ověření 
$ apt-get install apache2

# Ověření stavu:
$ systemctl status apache2

# Spustit při startu:
$ systemctl enable apache2

# Otevři v prohlížeči: http://localhost — zobrazí Apache výchozí stránku
// DŮLEŽITÉ CESTY Web root: /var/www/html/  ·  Konfigurace: /etc/apache2/  ·  Logy: /var/log/apache2/

Správa modulů

Moduly Apache2 začínají předponou libapache2-mod-. Aktivují se pomocí a2enmod, deaktivují pomocí a2dismod.
Aktivace / deaktivace modulů 
# Aktivace modulů:
$ a2enmod userdir
$ a2enmod rewrite
$ a2enmod ssl

# Deaktivace:
$ a2dismod userdir

# Reload po změně modulů:
$ sudo /etc/init.d/apache2 force-reload
# nebo:
$ systemctl reload apache2

Userdir modul — domovský adresář uživatele jako web

Modul userdir umožňuje každému systémovému uživateli zpřístupnit svůj adresář ~/public_html přes URL http://server/~uzivatel/.
Nastavení userdir 
# 1. Aktivovat modul:
$ a2enmod userdir
$ service apache2 restart

# 2. Vytvořit adresář a nastavit práva:
$ mkdir ~/public_html
$ chmod 755 ~/public_html

# 3. Vytvořit testovací stránku:
$ echo "<h1>Moje stránka</h1>" > ~/public_html/index.html

# 4. Test: http://localhost/~username/
// PHP V USERDIR PHP v userdir bývá ve výchozím stavu vypnuté. Oprav v /etc/apache2/mods-available/php8.x.conf — zakomentuj sekci s php_admin_value engine Off uvnitř bloku mod_userdir.c.

Virtual Hosts (subdomény)

Lokální subdoména pro vývoj 
# 1. Přidat do /etc/hosts:
127.0.0.1    mujweb.localhost

# 2. Vytvořit konfigurační soubor:
$ nano /etc/apache2/sites-available/mujweb.conf
/etc/apache2/sites-available/mujweb.conf 
<VirtualHost *:80>
  ServerName    mujweb.localhost
  DocumentRoot  /var/www/mujweb
  ErrorLog      ${APACHE_LOG_DIR}/mujweb-error.log
</VirtualHost>
Aktivace virtual hostu 
$ mkdir /var/www/mujweb
$ a2ensite mujweb.conf
$ systemctl reload apache2

Aktivace .htaccess

.htaccess soubor umožňuje konfiguraci Apache na úrovni adresáře (přesměrování, autentizace, URL rewrite...). Ve výchozím stavu je zakázán.
/etc/apache2/apache2.conf nebo site config 
# Změnit AllowOverride None na All v sekci <Directory /var/www>:
<Directory /var/www>
  Options Indexes FollowSymLinks MultiViews
  AllowOverride All          # změna z None na All
  Require all granted
</Directory>
Aktivace mod_rewrite (nutné pro .htaccess rewrite) 
$ a2enmod rewrite
$ systemctl reload apache2

Práva souborů webu

Nastavení práv pro /var/www 
# Rekurzivně nastavit čtení a zápis:
$ chmod -R 755 /var/www/html

# Pro vývojové prostředí (Apache user www-data):
$ chown -R www-data:www-data /var/www/html

# Přidat uživatele do skupiny www-data (pohodlnější správa):
$ usermod -a -G www-data $USER

phpMyAdmin

Instalace + zpřístupnění 
$ apt-get install phpmyadmin php-mbstring

# Pokud nefunguje http://localhost/phpmyadmin — vytvořit symlink:
$ ln -s /usr/share/phpmyadmin /var/www/html/phpmyadmin

$ systemctl reload apache2
