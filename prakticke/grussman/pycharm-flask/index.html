PyCharm + Flask

🐍 Python 🌶️ Flask formuláře a DB

1 — Nastavení PyCharm a virtuálního prostředí

PyCharm je výkonné IDE pro Python. Pro Flask aplikaci je doporučeno vytvořit virtuální prostředí (venv), které izoluje závislosti projektu.
Vytvoření virtuálního prostředí 
# Vytvoření venv
$ python3 -m venv venv

# Aktivace (Linux/macOS)
$ source venv/bin/activate

# Aktivace (Windows)
$ venv\Scripts\activate

# Instalace Flask
$ pip install flask flask-sqlalchemy

2 — Základy Flask frameworku

Minimální Flask aplikace 
# app.py
from flask import Flask, render_template, request, redirect, url_for

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/hello/<name>')
def hello(name):
    return f'Hello {name}!'

if __name__ == '__main__':
    app.run(debug=True)

3 — Formuláře ve Flasku

Zpracování formuláře 
# HTML formulář (templates/form.html)
<form method="POST" action="/submit">
  <input type="text" name="jmeno" placeholder="Jméno">
  <input type="email" name="email" placeholder="Email">
  <button type="submit">Odeslat</button>
</form>

# Python - zpracování (app.py)
@app.route('/submit', methods=['GET', 'POST'])
def submit():
    if request.method == 'POST':
        jmeno = request.form['jmeno']
        email = request.form['email']
        return f'Přijato: {jmeno}, {email}'
    return render_template('form.html')

4 — Databáze s Flask-SQLAlchemy

Model a připojení k DB 
from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///data.db'
db = SQLAlchemy(app)

# Definice modelu
class Uzivatel(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    jmeno = db.Column(db.String(80))
    email = db.Column(db.String(120))

# Vytvoření tabulek
with app.app_context():
    db.create_all()

# Vložení záznamu
novy = Uzivatel(jmeno='Jan', email='jan@test.cz')
db.session.add(novy)
db.session.commit()
