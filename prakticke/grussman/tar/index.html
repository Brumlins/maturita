tar — zálohování

🐧 Linux 📦 archivace & záloha

Parametry příkazu tar

ParametrFunkce
ccreate — vytvoří nový archiv
xextract — rozbalí archiv
vverbose — vypisuje zpracovávané soubory
ffile — specifikuje jméno archivu (vždy nutné)
zgzip komprese (.tar.gz)
jbzip2 komprese (.tar.bz2, lepší komprese, pomalejší)
C-C /cesta — cílový adresář pro rozbalení
tlist — zobrazí obsah archivu bez rozbalení

Vytvoření zálohy

Základní záloha adresáře 
# Syntaxe: tar cvfz [výstupní soubor] [co zálohovat]
$ tar cvfz /tmp/zaloha-etc.tar.gz /etc

# S časovým razítkem v názvu:
$ tar cvfz /tmp/zaloha-$(date +%Y%m%d).tar.gz /etc

# Záloha více adresářů najednou:
$ tar cvfz /backup/full.tar.gz /etc /home /var/www

# Zobrazení obsahu archivu (bez rozbalení):
$ tar tvfz /tmp/zaloha-etc.tar.gz

Rozbalení zálohy

Rozbalení archivu 
# Syntaxe: tar xvfz [soubor zálohy] -C [kam rozbalit]
$ tar xvfz /tmp/zaloha-etc.tar.gz -C /tmp/obnova/

# Rozbalení do aktuálního adresáře (vynech -C):
$ tar xvfz /tmp/zaloha.tar.gz

# Rozbalení jen konkrétního souboru z archivu:
$ tar xvfz zaloha.tar.gz etc/hostname

Inkrementální (rozdílová) záloha

Inkrementální záloha ukládá pouze změny od poslední zálohy — šetří místo a čas. Soubor .snar sleduje, které soubory se změnily.
Plná záloha + inkrementální záloha 
# Krok 1: PLNÁ záloha (první spuštění)
$ tar cvfz archive-full.tar.gz --listed-incremental=/var/log/backup.snar /home

# Krok 2: INKREMENTÁLNÍ záloha (přidá jen změny od kroku 1)
$ tar cvfz archive-incr1.tar.gz --listed-incremental=/var/log/backup.snar /home

# Obnovení: nejdřív plná záloha, pak inkrementální v pořadí
$ tar xvfz archive-full.tar.gz --listed-incremental=/dev/null -C /obnova/
$ tar xvfz archive-incr1.tar.gz --listed-incremental=/dev/null -C /obnova/

Zálohovací skript (maturita)

backup.sh — záloha /etc s logováním 
#!/bin/bash
datum=`date +%d%m%y`

tar cvfz /tmp/${datum}zaloha.tar.gz /etc

if [ "$?" -eq "0" ]; then
  logger "záloha OK"
else
  logger "záloha SELHALA"
fi
Záloha MySQL databáze 
#!/bin/bash
datum=`date +%d%m%y`

mysqldump -u root -p"heslo" --all-databases | gzip -c > /tmp/${datum}-mysql.gz

if [ "$?" -eq "0" ]; then
  logger "MySQL záloha OK"
else
  logger "MySQL záloha SELHALA"
fi
Spuštění skriptu 
# Vytvořit skript:
$ mcedit backup.sh

# Nastavit práva (spustitelný):
$ chmod +x backup.sh

# Spustit:
$ ./backup.sh
// CRON — automatické spouštění Pro pravidelné zálohy použij cron: crontab -e Např. denně ve 2:00: 0 2 * * * /root/backup.sh
