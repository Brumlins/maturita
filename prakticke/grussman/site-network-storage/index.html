Síťové úložiště

💾 Storage 🌐 sdílení dat v síti

1 — Typy síťových úložišť

Síťové úložiště umožňuje přístup k datům přes síť. Hlavní technologie: NFS (pro Linux/Unix), Samba/SMB (pro Windows i Linux), iSCSI (blokové zařízení přes IP síť).
// TIP NAS (Network Attached Storage) je zařízení, které poskytuje síťové úložiště, obvykle přes SMB nebo NFS.

2 — NFS (Network File System)

Nastavení NFS serveru 
# Instalace NFS serveru
$ apt-get install nfs-kernel-server

# Vytvoření sdíleného adresáře
$ mkdir -p /export/shared
$ chown -R nobody:nogroup /export/shared
$ chmod -R 755 /export/shared

# Konfigurace exportu (/etc/exports)
/export/shared 192.168.0.0/24(rw,sync,no_subtree_check)

# Aplikace změn
$ exportfs -a
$ systemctl restart nfs-kernel-server

# Na klientovi - připojení
$ mount -t nfs 192.168.0.10:/export/shared /mnt/shared

# Trvalé připojení (/etc/fstab)
192.168.0.10:/export/shared /mnt/shared nfs defaults 0 0

3 — Samba (SMB/CIFS)

Instalace a konfigurace Samby 
# Instalace Samby
$ apt-get install samba samba-common-bin

# Vytvoření sdíleného adresáře
$ mkdir -p /srv/samba/shared
$ chmod -R 0770 /srv/samba/shared

# Přidání uživatele do Samba
$ smbpasswd -a uzivatel

# Konfigurace (/etc/samba/smb.conf)
[shared]
comment = Sdílená složka
path = /srv/samba/shared
browseable = yes
read only = no
guest ok = no
valid users = uzivatel

# Restart
$ systemctl restart smbd nmbd

4 — iSCSI (Internet Small Computer Systems Interface)

iSCSI target a initiator 
# Na serveru (target) - instalace
$ apt-get install tgt

# Vytvoření iSCSI target (příklad s souborem)
$ dd if=/dev/zero of=/var/lib/iscsi/disk1.img bs=1M count=1024

# Konfigurace (/etc/tgt/conf.d/disk1.conf)

    backing-store /var/lib/iscsi/disk1.img
    initiator-address 192.168.0.0/24


$ systemctl restart tgt

# Na klientovi (initiator)
$ apt-get install open-iscsi
$ iscsiadm -m discovery -t sendtargets -p 192.168.0.10
$ iscsiadm -m node --login
