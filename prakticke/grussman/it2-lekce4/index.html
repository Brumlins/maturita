🌐 ISO/OSI 📡 TCP/IP model

1 — ISO/OSI model

ISO/OSI model (Open Systems Interconnection) je referenční model síťové komunikace rozdělený do 7 vrstev. Každá vrstva má specifickou funkci a komunikuje pouze s vrstvami nad a pod sebou.
7 vrstev OSI modelu 
7. Aplikační (Application)      - HTTP, FTP, SMTP, DNS
6. Prezentační (Presentation)  - SSL/TLS, kódování, komprese
5. Relační (Session)          - správa relací, RPC
4. Transportní (Transport)     - TCP, UDP, porty
3. Síťová (Network)           - IP, ICMP, směrování
2. Linková (Data Link)         - MAC, Ethernet, ARP
1. Fyzická (Physical)          - kabely, signály, bity

2 — TCP/IP model

TCP/IP model je praktický model používaný v reálném Internetu. Má 4 vrstvy a je jednodušší než ISO/OSI model.
4 vrstvy TCP/IP 
4. Aplikační    - HTTP, FTP, SMTP, DNS (odpovídá OSI 5-7)
3. Transportní   - TCP, UDP (stejné jako v OSI)
2. Síťová       - IP, ICMP (stejné jako v OSI)
1. Síťové rozhraní - Ethernet, MAC (odpovídá OSI 1-2)

3 — Srovnání obou modelů

Rozdíly a podobnosti 
# ISO/OSI (teoretický):
  + Přesný, 7 vrstev
  + Snadnější pochopení a výuka
  - Nereálný pro dnešní Internet

# TCP/IP (praktický):
  + Skutečně používaný na Internetu
  + Jednodušší struktura
  - Méně detailní rozdělení

# Encapsulace dat:
# Aplikace → Data → Segment (TCP) → Paket (IP) → Rámec (Ethernet) → Bity

4 — Detaily vrstev s příklady

Příklady protokolů podle vrstev 
# L7 Aplikační: HTTP/1.1, HTTP/2, HTTP/3 (QUIC)
#   FTP, SFTP, SMTP, POP3, IMAP
#   DNS, DHCP, SSH, Telnet

# L4 Transportní: TCP (spolehlivé), UDP (rychlé)
#   Porty: 80 (HTTP), 443 (HTTPS), 22 (SSH), 25 (SMTP)

# L3 Síťová: IPv4 (32-bit), IPv6 (128-bit)
#   Adresy: 192.168.0.1, 2001:db8::1

# L2 Linková: MAC adresy (48-bit)
#   Např. 00:1A:2B:3C:4D:5E
#   Přenosové médium: UTP, optika, WiFi
