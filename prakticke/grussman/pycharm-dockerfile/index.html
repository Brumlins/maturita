PyCharm a Dockerfile

🐍 Python 🐳 Docker v IDE

1 — Integrace PyCharm s Dockerem

PyCharm Professional podporuje přímou integraci s Dockerem. Můžete spravovat kontejnery, images a spouštět aplikace přímo v Docker kontejnerech z prostředí IDE.
// POZNÁMKA Docker integrace je dostupná v PyCharm Professional. Community verze vyžaduje externí konfiguraci.
Nastavení v PyCharm 
# 1. Settings → Build, Execution, Deployment → Docker
# 2. Přidat nový Docker server (obvykle automaticky detekován)
# 3. Ověřit připojení k Docker daemon

2 — Vytvoření Dockerfile v PyCharm

Dockerfile pro Python projekt 
FROM python:3.11-slim

# Nastavení pracovního adresáře
WORKDIR /app

# Kopírování souborů závislostí
COPY requirements.txt .

# Instalace závislostí
RUN pip install --no-cache-dir -r requirements.txt

# Kopírování celého projektu
COPY . .

# Expose port pro Flask
EXPOSE 5000

# Spuštění aplikace
CMD ["python", "app.py"]

3 — Spuštění aplikace v Docker kontejneru

Konfigurace spuštění v PyCharm 
# 1. Run → Edit Configurations
# 2. Přidat novou Docker konfiguraci
# 3. Vybrat Dockerfile a nastavit:
#    - Image tag: moje-flask-app
#    - Container name: flask-app
#    - Bind ports: 5000:5000
#    - Volume mapping: ./src:/app/src

# Ruční build a spuštění:
$ docker build -t flask-app .
$ docker run -p 5000:5000 flask-app

4 — Debugování v Docker kontejneru

PyCharm umožňuje remote debugování aplikací běžících v Dockeru pomocí nastavení Python interpreteru na Docker kontejner.
Nastavení interpreteru 
# 1. Settings → Project → Python Interpreter
# 2. Add Interpreter → Docker
# 3. Vybrat Docker image s Pythonem
# 4. Nastavit path mapping projektu
