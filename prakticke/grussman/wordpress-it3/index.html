WordPress v IT3

📝 WordPress 🖥️ instalace v učebně IT3

1 — Příprava LAMP stacku

V učebně IT3 je obvykle nutné nainstalovat LAMP (Linux, Apache, MySQL, PHP) nebo LEMP stack. WordPress vyžaduje PHP a MySQL databázi pro svůj běh.
Instalace komponent 
# Apache2
$ apt-get install apache2

# MySQL server
$ apt-get install mysql-server

# PHP a rozšíření
$ apt-get install php libapache2-mod-php php-mysql php-xml php-curl php-gd php-mbstring php-zip

# Restart Apache
$ systemctl restart apache2

2 — Stažení a rozbalení WordPressu

Stažení a nahrání 
# Stažení WordPressu
$ cd /var/www/html
$ wget https://cz.wordpress.org/latest-cz_CZ.tar.gz

# Rozbalení
$ tar -xzvf latest-cz_CZ.tar.gz

# Nastavení práv
$ chown -R www-data:www-data /var/www/html/wordpress
$ chmod -R 755 /var/www/html/wordpress

3 — Vytvoření databáze pro WordPress

MySQL příkazy 
# Přihlášení do MySQL
$ mysql -u root -p

# Vytvoření databáze
CREATE DATABASE wordpress;

# Vytvoření uživatele
CREATE USER 'wpuser'@'localhost' IDENTIFIED BY 'wppassword';

# Přidělení práv
GRANT ALL PRIVILEGES ON wordpress.* TO 'wpuser'@'localhost';
FLUSH PRIVILEGES;
EXIT;

# Konfigurace WordPress
$ cp /var/www/html/wordpress/wp-config-sample.php /var/www/html/wordpress/wp-config.php
$ nano /var/www/html/wordpress/wp-config.php
# Upravit DB_NAME, DB_USER, DB_PASSWORD

4 — Dokončení instalace v prohlížeči

Po přípravě databáze a souborů otevřete v prohlížeči adresu serveru s adresářem wordpress a dokončete instalaci přes webové rozhraní.
Přístup a nastavení 
# V prohlížeči:
# http://localhost/wordpress

# Nebo pokud je to na jiném stroji:
# http://IP_ADRESA/wordpress

# Zvolit jazyk, název webu, admin účet
# Po dokončení: přihlášení do /wp-admin/
