MAC Adresy

🔗 MAC 🌐 vztah k IP adresám

1 — Co je MAC adresa

MAC adresa (Media Access Control) je fyzická adresa síťového rozhraní. Je to unikátní identifikátor přiřazený výrobcem, který se používá na linkové vrstvě (L2) pro komunikaci v lokální síti.
// TIP MAC adresa má 48 bitů (6 bajtů) a je obvykle zapsána hexadecimálně, např. 00:1A:2B:3C:4D:5E.

2 — Formát a struktura MAC adresy

Rozbor MAC adresy 
# Příklad: 00:1A:2B:3C:4D:5E
# První 3 bajty (00:1A:2B) = OUI (Organizationally Unique Identifier)
#   Určuje výrobce síťové karty
# Poslední 3 bajty (3C:4D:5E) = individuální část (NIC Specific)

# Zjištění výrobce:
# http://macvendors.com/ nebo vyhledat v databázi

# Bit 2 v prvním bajtu:
#   0 = Unicast (individuální)
#   1 = Multicast (skupinová)

# MAC adresa se někdy nazývá hardware address nebo fyzická adresa

3 — Vztah MAC adresy k IP adrese

Zatímco MAC adresa funguje na linkové vrstvě (L2) a slouží pro komunikaci v lokální síti, IP adresa funguje na síťové vrstvě (L3) a umožňuje směrování mezi sítěmi.
ARP - překlad IP na MAC 
# Když počítač A (192.168.0.10) chce poslat data na B (192.168.0.20):
# 1. A zkontroluje, zda je B ve stejné podsíti
# 2. A se podívá do ARP tabulky, zda zná MAC adresu B
# 3. Pokud ne, pošle ARP request (broadcast): "Kdo má IP 192.168.0.20?"
# 4. B odpoví ARP reply: "Já mám tuto IP, moje MAC je XX:XX:XX:XX:XX:XX"
# 5. A uloží tuto informaci do ARP cache

# Zobrazení ARP tabulky:
$ arp -a
$ ip neigh show
Rozdíl mezi MAC a IP 
# MAC adresa:
#   - Fyzická, permanentní (nelze změnit, ale dá se spoofovat)
#   - Pracuje v lokální síti (L2)
#   - 48 bitů, hexadecimální zápis

# IP adresa:
#   - Logická, měnitelná (přiděluje správce sítě)
#   - Pracuje globálně (L3), umožňuje směrování
#   - IPv4 = 32 bitů, IPv6 = 128 bitů

4 — Příkazy pro práci s MAC adresami

Linux příkazy 
# Zobrazení MAC adresy rozhraní
$ ip link show
$ ifconfig  # (starší)

# Změna MAC adresy (dočasná, po restartu se vrátí)
$ ip link set eth0 down
$ ip link set eth0 address 00:11:22:33:44:55
$ ip link set eth0 up

# ARP tabulka
$ ip neigh
$ arp -n

# Vymazání ARP záznamu
$ ip neigh del 192.168.0.20 dev eth0
