Django CMS

🐍 Django 📝 Content Management System

1 — Příprava prostředí

Django CMS je systém pro správu obsahu postavený na frameworku Django. Vyžaduje Python 3.8+, Django a databázi (SQLite pro vývoj, PostgreSQL pro produkci).
Vytvoření virtuálního prostředí 
# Vytvoření adresáře projektu
$ mkdir django-cms-project
$ cd django-cms-project

# Vytvoření a aktivace venv
$ python3 -m venv venv
$ source venv/bin/activate

# Upgrade pip
$ pip install --upgrade pip

2 — Instalace Django CMS

Instalace balíčků 
# Instalace Django CMS a jeho závislostí
$ pip install django-cms djangocms-admin-style

# Vytvoření Django projektu
$ django-admin startproject myproject .

# Vytvoření CMS aplikace
$ python manage.py startapp mycms

# Instalované balíčky
$ pip freeze > requirements.txt

3 — Konfigurace projektu

settings.py úpravy 
# myproject/settings.py
INSTALLED_APPS = [
    'django.contrib.sites',
    'cms',
    'menus',
    'treebeard',
    'djangocms_admin_style',
    'djangocms_text_ckeditor',
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
]

SITE_ID = 1
LANGUAGE_CODE = 'cs'

# Databáze (výchozí SQLite)
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}

4 — Migrace a spuštění

Inicializace CMS 
# Migrace databáze
$ python manage.py migrate

# Vytvoření superuživatele
$ python manage.py createsuperuser

# Spuštění vývojového serveru
$ python manage.py runserver 0.0.0.0:8000

# Přístup v prohlížeči:
# http://localhost:8000
# Administrace: http://localhost:8000/admin/
