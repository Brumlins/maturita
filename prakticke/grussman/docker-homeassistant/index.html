Docker a Home Assistant

🏠 Home Assistant 🐳 v Docker kontejneru

1 — Co je Home Assistant

Home Assistant je open-source platforma pro domácí automatizaci. Umožňuje propojit a ovládat chytrá zařízení z jednoho místa, podporuje stovky integrací a běží lokálně bez nutnosti cloudu.

2 — Instalace Home Assistant přes Docker

Spuštění Home Assistant kontejneru 
# Vytvoření adresáře pro konfiguraci
$ mkdir -p /home/uzivatel/homeassistant

# Spuštění Home Assistant
$ docker run -d \
  --name homeassistant \
  --restart=unless-stopped \
  -v /home/uzivatel/homeassistant:/config \
  -v /etc/localtime:/etc/localtime:ro \
  --network=host \
  ghcr.io/home-assistant/home-assistant:stable

# Přístup přes prohlížeč:
# http://IP_ADRESA:8123
// TIP Parametr --network=host je pro Home Assistant často nutný kvůli autodiscovery zařízení v síti.

3 — Základní konfigurace

configuration.yaml 
# /config/configuration.yaml
homeassistant:
  name: Můj Dům
  latitude: 50.0
  longitude: 14.0
  elevation: 300
  unit_system: metric
  time_zone: Europe/Prague

default_config:

logger:
  default: info

4 — Přidání zařízení a integrace

Home Assistant podporuje automatické objevování zařízení (zeroconf, UPnP, Bluetooth). Další integrace lze přidat přes webové rozhraní v sekci Nastavení → Zařízení a služby.
Příklad MQTT integrace 
# configuration.yaml
mqtt:
  broker: 192.168.0.10
  port: 1883
  username: uzivatel
  password: heslo

# Restart Home Assistant po změně konfigurace:
$ docker restart homeassistant
