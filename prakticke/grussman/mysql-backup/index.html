MySQL Backup

🗄️ MySQL 💾 zálohování a obnova

1 — Zálohování pomocí mysqldump

mysqldump je nástroj pro vytvoření logické zálohy MySQL databáze. Vytváří SQL soubor se všemi příkazy potřebnými pro obnovu databáze.
Základní záloha 
# Záloha konkrétní databáze
$ mysqldump -u root -p moje_db > backup_$(date +%Y%m%d).sql

# Záloha všech databází
$ mysqldump -u root -p --all-databases > all_backup.sql

# Záloha s kompresí (gzip)
$ mysqldump -u root -p moje_db | gzip > backup.sql.gz

# Záloha struktury bez dat
$ mysqldump -u root -p --no-data moje_db > struktura.sql

2 — Obnova databáze

Obnova ze zálohy 
# Vytvoření prázdné databáze
$ mysql -u root -p -e "CREATE DATABASE moje_db;"

# Obnova ze SQL souboru
$ mysql -u root -p moje_db < backup.sql

# Obnova z komprimované zálohy
$ gunzip < backup.sql.gz | mysql -u root -p moje_db

# Obnova všech databází
$ mysql -u root -p < all_backup.sql

3 — Automatizace záloh přes cron

Skript pro automatické zálohy 
#!/bin/bash
# /root/scripts/mysql-backup.sh
BACKUP_DIR="/var/backups/mysql"
DATE=$(date +%Y%m%d)
USER="backup_user"
PASS="heslo"

mkdir -p $BACKUP_DIR

# Záloha všech DB
mysqldump -u $USER -p$PASS --all-databases | gzip > $BACKUP_DIR/all_$DATE.sql.gz

# Mazání záloh starších 30 dní
find $BACKUP_DIR -name "*.sql.gz" -mtime +30 -delete

# Přidání do crontab (každý den 2:00)
# 0 2 * * * /root/scripts/mysql-backup.sh

4 — Tipy a triky

Pokročilé možnosti 
# Záloha s transakcemi (InnoDB)
$ mysqldump -u root -p --single-transaction moje_db > backup.sql
