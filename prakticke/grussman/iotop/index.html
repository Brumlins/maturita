iotop — monitoring I/O

🐧 Linux 💾 disk I/O monitoring

Instalace

Instalace iotop 
$ apt-get install iotop

# Vyžaduje kernel s CONFIG_TASK_IO_ACCOUNTING
# Všechny moderní Ubuntu/Debian kernely ho mají

Spuštění

Základní spuštění 
# Musí běžet jako root:
$ sudo iotop

# Zobrazit pouze procesy s I/O aktivitou:
$ sudo iotop -o

# Neinteraktivní výstup (pro scripty):
$ sudo iotop -b -n 5

# Sledovat konkrétní PID:
$ sudo iotop -p 1234

Čtení výstupu

iotop zobrazuje procesy seřazené podle I/O aktivity, podobně jako top pro CPU.
SloupecVýznam
TIDThread ID procesu
PRIOI/O priorita (be/rt + číslo)
USERvlastník procesu
DISK READaktuální rychlost čtení z disku
DISK WRITEaktuální rychlost zápisu na disk
SWAPINčas strávený čekáním na swap
IO% času čekání na I/O operace
// DISK READ vs WRITE Celkový read/write je zobrazen v záhlaví obrazovky — vidíš celkové zatížení disku systémem.
