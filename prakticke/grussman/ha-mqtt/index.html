📡 MQTT 🏠 integrace s HA

1 — Co je MQTT

MQTT (Message Queuing Telemetry Transport) je lehký protokol pro komunikaci IoT zařízení. Funguje na principu publikování a odběru zpráv (publish/subscribe) přes centrální broker.
// TIP MQTT používá TCP port 1883 (nešifrované) nebo 8883 (TLS). Je ideální pro zařízení s omezenými zdroji.
Koncepty MQTT 
# Broker (server) - zprostředkovatel zpráv
# Publisher (vydavatel) - odesílá zprávy na téma (topic)
# Subscriber (odběratel) - přijímá zprávy z témat
# Topic (téma) - adresa zprávy, např. "dum/obývák/teplota"
# QoS (Quality of Service) - 0, 1, 2 (spolehlivost doručení)

2 — Instalace Mosquitto brokeru

Mosquitto v Dockeru 
# Spuštění Eclipse Mosquitto
$ docker run -d \
  --name mosquitto \
  -p 1883:1883 \
  -p 9001:9001 \
  -v /home/uzivatel/mosquitto/config:/mosquitto/config \
  -v /home/uzivatel/mosquitto/data:/mosquitto/data \
  -v /home/uzivatel/mosquitto/log:/mosquitto/log \
  eclipse-mosquitto

# Test publikování zprávy
$ mosquitto_pub -h localhost -t "test/topic" -m "Ahoj MQTT"

# Test odběru zpráv
$ mosquitto_sub -h localhost -t "test/#"

3 — MQTT integrace v Home Assistant

Konfigurace MQTT v HA 
# configuration.yaml
mqtt:
  broker: 192.168.0.10
  port: 1883
  discovery: true
  discovery_prefix: homeassistant

# Příklad senzoru přes MQTT
sensor:
  - platform: mqtt
    name: "Teplota obývák"
    state_topic: "dum/obivak/teplota"
    unit_of_measurement: "°C"
    value_template: "{{ value_json.temperature }}"

4 — Příklad: ESP32 publikuje do MQTT

ESP32 s MicroPython a MQTT 
# main.py na ESP32
import time
from umqtt.simple import MQTTClient

SERVER = b'192.168.0.10'
TOPIC = b'dum/obivak/teplota'

client = MQTTClient("esp32", SERVER)
client.connect()

while True:
    teplota = 22.5  # čtení z senzoru
    client.publish(TOPIC, str(teplota))
    time.sleep(60)
