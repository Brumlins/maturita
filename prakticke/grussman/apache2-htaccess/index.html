Apache2 a .htaccess

🌐 Apache2 📄 .htaccess, CGI, práva

1 — Instalace a základní nastavení Apache2

Instalace na Debian/Ubuntu 
# Instalace Apache2
$ apt-get install apache2

# Spuštění a povolení
$ systemctl start apache2
$ systemctl enable apache2

# Test v prohlížeči:
# http://localhost

# Hlavní konfigurační soubory:
# /etc/apache2/apache2.conf - hlavní konfigurace
# /etc/apache2/sites-available/ - dostupné weby
# /var/www/html/ - výchozí DocumentRoot

2 — .htaccess soubory

Soubor .htaccess umožňuje konfiguraci Apache na úrovni adresářů bez nutnosti editace hlavního konfiguračního souboru. Musí být povolen v hlavní konfiguraci (AllowOverride).
Příklady .htaccess pravidel 
# Povolení .htaccess v Apache konfiguraci
# /etc/apache2/sites-available/000-default.conf
<Directory /var/www/html>
    AllowOverride All
</Directory>

# .htaccess - přesměrování na HTTPS
RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

# Ochrana adresáře heslem
AuthType Basic
AuthName "Chráněná sekce"
AuthUserFile /etc/apache2/.htpasswd
Require valid-user

# Vytvoření uživatele pro .htpasswd
$ htpasswd -c /etc/apache2/.htpasswd uzivatel

3 — CGI-bin (Common Gateway Interface)

Nastavení CGI v Apache2 
# Povolení CGI modulu
$ a2enmod cgid
$ systemctl restart apache2

# Příklad CGI skriptu (Python)
# /usr/lib/cgi-bin/hello.py
#!/usr/bin/python3
print("Content-type: text/html")
print()
print("<h1>Hello from CGI!</h1>")

# Nastavení práv pro CGI skript
$ chmod +x /usr/lib/cgi-bin/hello.py

# Test v prohlížeči:
# http://localhost/cgi-bin/hello.py

4 — Práva souborů a vlastnictví

Správa práv pro webové soubory 
# Nastavení vlastníka na www-data (uživatel Apache)
$ chown -R www-data:www-data /var/www/html

# Práva pro adresáře (755)
$ find /var/www/html -type d -exec chmod 755 {} \;

# Práva pro soubory (644)
$ find /var/www/html -type f -exec chmod 644 {} \;

# Vysvětlení práv:
# 7 = rwx (read, write, execute) pro vlastníka
# 5 = r-x pro skupinu
# 5 = r-x pro ostatní
