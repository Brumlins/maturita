SSH — Secure Shell

SSH (Secure Shell) umožňuje bezpečný vzdálený přístup k serveru. Používá šifrování pro ochranu hesla i dat.
1 — Instalace SSH serveru
Debian/Ubuntu
$ apt-get install openssh-server

# Ověření stavu:
$ systemctl status ssh
$ systemctl enable ssh   # spustit při startu
2 — Generování SSH klíče
SSH klíče fungují na principu asymetrické kryptografie: privátní klíč zůstane u tebe, veřejný klíč nahraješ na server.
Generování klíče (RSA 4096)
$ ssh-keygen -t rsa -b 4096
# Zeptá se na umístění (výchozí ~/.ssh/id_rsa) a passphrase
# Výsledek:
#   ~/.ssh/id_rsa       ← privátní klíč (NIKDY neposílat!)
#   ~/.ssh/id_rsa.pub   ← veřejný klíč (jde nahrát na servery)
3 — Kopírování klíče na server
Metoda 1 — ssh-copy-id
$ ssh-copy-id uzivatel@192.168.0.10
4 — Zákaz přihlášení heslem
/etc/ssh/sshd_config
PasswordAuthentication no
PermitRootLogin no
PubkeyAuthentication yes
$ systemctl restart ssh
SSH klíče a přihlášení bez hesla

Bash skripty

Bash skripty slouží k automatizaci úloh v Linuxu. Jsou nezbytné pro maturitní zkoušku.
1 — Vytvoření a spuštění
Základní struktura
#!/bin/bash
jmeno="Alice"
echo "Ahoj $jmeno"

# Nastavit spustitelný:
$ chmod +x skript.sh
$ ./skript.sh
2 — Podmínky a cykly
if / else
if test -e "/tmp/zaloha"; then
  echo "adresář existuje"
else
  mkdir /tmp/zaloha
fi
3 — Zálohovací skript (maturita)
Záloha /etc
datum=$(date +%Y-%m-%d)
tar cvfz /backup/etc_$datum.tar.gz /etc
$?  # návratová hodnota (0 = OK)

IP adresa je logická adresa pro identifikaci v síti.
