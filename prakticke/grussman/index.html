SSH — Secure Shell

SSH umožňuje bezpečný vzdálený přístup k serveru.
1 — Instalace SSH serveru
Debian/Ubuntu
$ apt-get install openssh-server

# Ověření stavu:
$ systemctl status ssh
$ systemctl enable ssh
2 — Generování SSH klíče
Generování klíče (RSA 4096)
$ ssh-keygen -t rsa -b 4096
SSH klíče a přihlášení bez hesla

Bash skripty

Bash skripty slouží k automatizaci úloh v Linuxu.
1 — Vytvoření a spuštění
Základní struktura
#!/bin/bash
jmeno="Alice"
echo "Ahoj $jmeno"

# Nastavit spustitelný:
$ chmod +x skript.sh
$ ./skript.sh
