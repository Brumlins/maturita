Dynamický routing

🌐 Routing 🔄 automatické směrování

1 — Co je dynamický routing

Dynamický routing umožňuje směrovačům automaticky učit se a vyměňovat si informace o cestách v síti. Při výpadku spoje dokáže najít alternativní cestu.
// TIP Oproti statickému routingu (ruční zápis cest) se dynamický routing hodí pro větší sítě.

2 — Hlavní protokoly dynamického routingu

Přehled protokolů 
# IGP (Interior Gateway Protocols) - uvnitř autonomního systému:
#   RIP (Routing Information Protocol) - starší, metrika = počet hopů
#   OSPF (Open Shortest Path First) - moderní, metrika = šířka pásma
#   EIGRP (Cisco proprietární)

# EGP (Exterior Gateway Protocols) - mezi autonomními systémy:
#   BGP (Border Gateway Protocol) - Internet backbone

# Rozdíl ve výpočtu cesty:
#   Distance Vector (RIP) - sděluje "kam" a "jak daleko"
#   Link-State (OSPF) - zná celou topologii sítě

3 — OSPF (Open Shortest Path First)

Konfigurace OSPF na Cisco (příklad) 
# Vstup do konfiguračního módu
Router> enable
Router# configure terminal

# Zapnutí OSPF procesu (číslo 1 = process ID)
Router(config)# router ospf 1

# Definice sítí, které budou inzerovány (area 0 = backbone)
Router(config-router)# network 192.168.1.0 0.0.0.255 area 0
Router(config-router)# network 10.0.0.0 0.255.255.255 area 1

# Zobrazení OSPF sousedů
Router# show ip ospf neighbor

# Zobrazení routovací tabulky
Router# show ip route ospf

4 — RIP (Routing Information Protocol)

RIP konfigurace (Cisco) 
# Zapnutí RIP
Router(config)# router rip

# Verze RIP (2 je modernější, podporuje CIDR)
Router(config-router)# version 2
Router(config-router)# no auto-summary

# Definice sítí
Router(config-router)# network 192.168.1.0
Router(config-router)# network 10.0.0.0

# Zobrazení RIP rout
Router# show ip route rip

# RIP omezení: max 15 hopů (16 = nedosažitelné)
# Časový interval: 30 sekund mezi aktualizacemi
