🐙 Git 🔄 správa verzí (interaktivní kurz)

1 — Co je Git

Git je distribuovaný systém pro správu verzí (VCS). Umožňuje sledovat změny v kódu, vracet se k předchozím verzím a spolupracovat s dalšími vývojáři.
// TIP Doporučuji interaktivní kurz: Learn Git Branching

2 — Základní příkazy

První kroky s Gitem 
# Konfigurace (prvně po instalaci)
$ git config --global user.name "Tvé Jméno"
$ git config --global user.email "tvuj@email.cz"

# Inicializace nového repozitáře
$ git init

# Přidání souborů do staging area
$ git add soubor.txt
$ git add .  # přidá všechny změněné soubory

# Commit (uložení změn)
$ git commit -m "Popis změn"

# Stav repozitáře
$ git status

# Historie commitů
$ git log --oneline

3 — Větve (branches)

Práce s větvemi 
# Výpis větví
$ git branch

# Vytvoření nové větve
$ git branch nove-funkce

# Přepnutí na větev
$ git checkout nove-funkce
# nebo nověji:
$ git switch nove-funkce

# Vytvoření a přepnutí (zkratka)
$ git checkout -b nove-funkce

# Sloučení větve do aktuální
$ git merge nove-funkce

# Smazání větve
$ git branch -d nove-funkce

4 — Vzdálené repozitáře (GitHub, GitLab)

Práce s remote 
# Přidání vzdáleného repozitáře
$ git remote add origin https://github.com/uzivatel/repo.git

# Odeslání změn na server (push)
$ git push -u origin main

# Stažení změn ze serveru (pull = fetch + merge)
$ git pull

# Naklonování existujícího repozitáře
$ git clone https://github.com/uzivatel/repo.git

# Zobrazení vzdálených repozitářů
$ git remote -v
