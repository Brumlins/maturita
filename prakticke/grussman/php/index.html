1 — Instalace PHP s Apache2

Instalace PHP na Linuxu 
# Instalace PHP a modulu pro Apache
$ apt-get install php libapache2-mod-php php-mysql

# Restart Apache
$ systemctl restart apache2

# Test PHP - vytvořit /var/www/html/info.php
<?php phpinfo(); ?>

# Test v prohlížeči: http://localhost/info.php
# (Poté smazat info.php z bezpečnostních důvodů!)

2 — Základní syntaxe PHP

Základní konstrukce 
<?php
// Komentáře
$jmeno = "Jan";  // proměnná začíná $
$vek = 20;

// Výpis
echo "Ahoj $jmeno!";
print("Věk: $vek");

// Podmínky
if ($vek >= 18) {
    echo "Dospělý";
} else {
    echo "Mladistvý";
}

// Cyklus
for ($i = 0; $i < 5; $i++) {
    echo $i;
}
?>

3 — Formuláře a zpracování POST/GET

Zpracování formuláře v PHP 
<!-- HTML formulář -->
<form method="POST" action="zpracovani.php">
  <input type="text" name="jmeno">
  <input type="submit" value="Odeslat">
</form>

<?php
// zpracovani.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $jmeno = $_POST["jmeno"];
    echo "Přijato: $jmeno";
}
?>

# Přístup k GET parametrům: $_GET["parametr"]
# Přístup k obojímu: $_REQUEST["parametr"]

4 — Připojení k MySQL databázi

MySQLi příklad 
<?php
$servername = "localhost";
$username = "root";
$password = "heslo";
$dbname = "moje_db";

// Vytvoření připojení
$conn = new mysqli($servername, $username, $password, $dbname);

// Kontrola připojení
if ($conn->connect_error) {
    die("Připojení selhalo: " . $conn->connect_error);
}

// Dotaz
$sql = "SELECT id, jmeno FROM uzivatele";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
        echo "id: " . $row["id"]. " - Jméno: " . $row["jmeno"]. "<br>";
    }
}
$conn->close();
?>
