IT2 Lekce 10

☁️ Cloud 💻 výpočetní modely

1 — Výpočetní modely

Výpočetní modely popisují způsob, jakým jsou poskytovány výpočetní zdroje. Hlavní modely jsou: On-premise, IaaS, PaaS, SaaS a FaaS (serverless).

2 — Typy výpočetních modelů

Přehled modelů 
# On-premise (vlastní infrastruktura)
# - Kupujete hardware, máte plnou kontrolu
# - Vysoké počáteční náklady, vlastní údržba

# IaaS (Infrastructure as a Service)
# - Virtuální servery, sítě, úložiště (AWS EC2, Azure VM)
# - Vy spravujete OS a aplikace

# PaaS (Platform as a Service)
# - Platforma pro vývoj a běh aplikací (Heroku, Google App Engine)
# - Nemusíte řešit OS, jen aplikaci

# SaaS (Software as a Service)
# - Hotové aplikace přes prohlížeč (Gmail, Office 365)
# - Žádná správa, jen používání

# FaaS (Function as a Service / Serverless)
# - Spouštění funkcí na vyžádání (AWS Lambda)
# - Platíte jen za čas běhu funkce

3 — Cloud computing

Cloud computing poskytuje výpočetní zdroje přes Internet. Hlavní poskytovatelé: AWS (Amazon), Microsoft Azure, Google Cloud Platform (GCP).
Typy cloudu 
# Public Cloud - veřejný, sdílený hardware
# Private Cloud - soukromý, jen pro jednu organizaci
# Hybrid Cloud - kombinace veřejného a soukromého
# Multi-cloud - využití více poskytovatelů

# Výhody: škálovatelnost, flexibilita, platba za spotřebu
# Nevýhody: závislost na poskytovateli, bezpečnost dat

4 — Srovnání modelů

Kdy který model použít 
# On-premise:
#   - Citlivá data, regulované odvětví
#   - Stálé a předvídatelné zátěže

# IaaS:
#   - Migrace existujících aplikací do cloudu
#   - Potřeba plné kontroly nad OS

# PaaS:
#   - Vývoj nových aplikací
#   - Rychlé prototypování

# SaaS:
#   - Běžné kancelářské aplikace
#   - CRM, ERP systémy
