Linux příkazy (IT4)

🐧 Linux ⌨️ přehled příkazů

1 — Systém a správa procesů

Systémové příkazy 
# Informace o systému
$ uname -a  # jádro a verze systému
$ lsb_release -a  # distribuce
$ uptime  # doba běhu a zátěž

# Procesy
$ ps aux  # všechny procesy
$ top  # interaktivní přehled (q = quit)
$ htop  # modernější top (nutno instalovat)

# Správa služeb (systemd)
$ systemctl status nginx
$ systemctl start/stop/restart služba
$ journalctl -u služba -f  # logy služby

# Vypnutí/restart
$ shutdown -h now  # vypnutí
$ reboot  # restart

2 — Síťové příkazy

Síťová konfigurace a diagnostika 
# Rozhraní a IP adresy
$ ip addr show  # (nahrazuje ifconfig)
$ ip link set eth0 up/down
$ ip route show  # routovací tabulka

# Konfigurace IP (dočasná)
$ ip addr add 192.168.0.10/24 dev eth0

# Testování sítě
$ ping 8.8.8.8
$ traceroute google.com
$ mtr google.com  # kombinace ping+traceroute

# DNS
$ dig google.com
$ nslookup google.com

# Síťové služby
$ ss -tuln  # naslouchající porty (nahrazuje netstat)
$ nmap -sT localhost  # sken portů

3 — Disky, soubory a úložiště

Práce s disky a soubory 
# Využití disku
$ df -h  # volné místo (human readable)
$ du -sh /home  # velikost adresáře

# Připojování disků
$ mount /dev/sdb1 /mnt/data
$ umount /mnt/data
$ lsblk  # seznam blokových zařízení

# Práce se soubory
$ tar -czvf archiv.tar.gz /data  # komprese
$ tar -xzvf archiv.tar.gz  # dekomprese
$ rsync -av /zdroj/ /cil/  # synchronizace

# Hledání
$ find /home -name "*.txt"
$ grep -r "text" /home

4 — Uživatelé, skupiny a práva

Správa uživatelů 
# Uživatelé
$ adduser novy_uzivatel
$ passwd uzivatel  # změna hesla
$ deluser uzivatel
$ whoami  # aktuální uživatel
$ id uzivatel  # UID, GID, skupiny

# Skupiny
$ addgroup nova_skupina
$ adduser uzivatel nova_skupina

# Práva souborů
$ chmod 755 soubor  # rwxr-xr-x
$ chown uzivatel:skupina soubor
$ chmod -R 755 /data  # rekurzivně

# Sudo
$ visudo  # editace /etc/sudoers (bezpečně)
# Přidání: uzivatel ALL=(ALL:ALL) ALL
