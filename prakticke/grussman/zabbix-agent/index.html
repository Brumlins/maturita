Zabbix Agent

📊 Monitoring 🔍 Zabbix agent

1 — Co je Zabbix

Zabbix je open-source řešení pro monitorování sítí, serverů a aplikací. Skládá se ze serveru, databáze a agentů, kteří sbírají data ze sledovaných zařízení. Zabbix agent je program běžící na monitorovaném stroji.

2 — Instalace Zabbix agenta

Instalace na Debian/Ubuntu 
# Přidání repozitáře Zabbix
$ wget https://repo.zabbix.com/zabbix/6.0/ubuntu/pool/main/z/zabbix-release/zabbix-release_6.0-4+ubuntu22.04_all.deb
$ dpkg -i zabbix-release_6.0-4+ubuntu22.04_all.deb
$ apt-get update

# Instalace agenta
$ apt-get install zabbix-agent2

# Spuštění a povolení při bootu
$ systemctl enable zabbix-agent2
$ systemctl start zabbix-agent2

3 — Konfigurace agenta

/etc/zabbix/zabbix_agent2.conf 
# Adresa Zabbix serveru
Server=192.168.0.10

# Adresa pro aktivní checky
ServerActive=192.168.0.10

# Název hosta (musí odpovídat v Zabbix webu)
Hostname=muj-server

# Port (výchozí 10050)
ListenPort=10050

# Restart agenta po změně konfigurace
$ systemctl restart zabbix-agent2
// TIP Zabbix agent2 je novější verze s lepší podporou pro různé protokoly a paralelní zpracování.

4 — Položky (Items) a šablony

V Zabbix webovém rozhraní přidáte hosta a přiřadíte mu šablony (Templates), které obsahují předdefinované položky, triggery a grafy.
Příklad vlastní položky (UserParameter) 
# V /etc/zabbix/zabbix_agent2.conf přidat:
UserParameter=system.uptime.days,uptime | awk '{print int($1/86400)}'
UserParameter=http.check[*],curl -s -o /dev/null -w "%{http_code}" $1

# Test položky na agentovi:
$ zabbix_agent2 -t system.uptime.days
