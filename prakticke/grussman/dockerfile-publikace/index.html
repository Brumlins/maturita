Dockerfile Publikace

🐳 Docker 📦 publikace aplikací

1 — Co je Dockerfile

Dockerfile je textový soubor obsahující instrukce pro sestavení Docker image. Definuje základní image, kopírované soubory, instalované balíčky a spouštěné příkazy.
// TIP Dockerfile umožňuje automatizovat vytváření image a zajistit konzistentní prostředí napříč vývojem a produkcí.

2 — Základní syntaxe Dockerfile

Hlavní instrukce 
# FROM - základní image
FROM ubuntu:22.04

# RUN - spuštění příkazu při buildování
RUN apt-get update && apt-get install -y nginx

# COPY - zkopírování souborů do image
COPY ./app /var/www/app

# WORKDIR - nastavení pracovního adresáře
WORKDIR /var/www/app

# EXPOSE - informace o naslouchajícím portu
EXPOSE 80

# CMD - výchozí příkaz při spuštění kontejneru
CMD ["nginx", "-g", "daemon off;"]

# ENV - nastavení proměnných prostředí
ENV APP_ENV=production

3 — Příklad Dockerfile pro Python aplikaci

Dockerfile pro Flask aplikaci 
FROM python:3.11-slim

WORKDIR /app

# Kopírování souborů závislostí
COPY requirements.txt .

# Instalace závislostí
RUN pip install --no-cache-dir -r requirements.txt

# Kopírování zdrojového kódu
COPY . .

# Expose port
EXPOSE 5000

# Spuštění aplikace
CMD ["python", "app.py"]

4 — Build a publikace image

Sestavení a nahrání na Docker Hub 
# Build image ze současného adresáře
$ docker build -t moje-app:1.0 .

# Označení image pro Docker Hub
$ docker tag moje-app:1.0 uzivatel/moje-app:1.0

# Přihlášení k Docker Hub
$ docker login

# Nahrání image na Docker Hub
$ docker push uzivatel/moje-app:1.0

# Stažení image z Docker Hub (kdekoli jinde)
$ docker pull uzivatel/moje-app:1.0
