MicroPython ESP

🐍 MicroPython 📟 ESP32 a ESP8266

1 — Co je MicroPython

MicroPython je implementace Pythonu 3 optimalizovaná pro mikrokontroléry. Umožňuje programovat ESP32/ESP8266 v Pythonu namísto C++ (Arduino) nebo Lua (NodeMCU).
// TIP MicroPython má vlastní REPL (interaktivní shell) přes sériovou linku, což umožňuje okamžité testování kódu.

2 — Nahrání MicroPython firmware na ESP

Pomocí esptool 
# Instalace esptool
$ pip install esptool

# Stažení firmware (z micropython.org/download)
# Např. pro ESP32: esp32-20220618-v1.19.1.bin

# Vymazání flash
$ esptool.py --port /dev/ttyUSB0 erase_flash

# Nahraní firmware
$ esptool.py --port /dev/ttyUSB0 --baud 460800 write_flash -z 0x1000 esp32-20220618-v1.19.1.bin

# Přístup přes sériovou konzoli
$ screen /dev/ttyUSB0 115200
# nebo
$ minicom -D /dev/ttyUSB0 -b 115200

4 — Příklady kódu pro ESP32

main.py - blink a WiFi 
# main.py - uložit do ESP32
import machine
import time
import network

# LED blink (vestavěná LED na pinu 2)
led = machine.Pin(2, machine.Pin.OUT)

# Připojení k WiFi
wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect("SSID", "heslo")

while not wlan.isconnected():
    time.sleep(1)
    print("Připojuji...")

print("Připojeno:", wlan.ifconfig())

# Cyklus blikání
while True:
    led.value(1)
    time.sleep(0.5)
    led.value(0)
    time.sleep(0.5)
