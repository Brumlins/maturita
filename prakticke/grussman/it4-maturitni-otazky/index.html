Maturitní otázky (TVY)

📝 Maturita 🎓 technická vybavenost

1 — TVY (Technická vybavenost)

Část maturitní zkoušky z odborných předmětů, kde student prokazuje znalosti z hardwaru, sítí, operačních systémů a dalších technických oblastí.
// TIP Důležité je umět prakticky předvést konfigurace, nejen teoreticky znát.

2 — Hlavní témata pro maturitu

Přehled oblastí 
# 1. SÍTĚ A PROTOKOLY
#    - ISO/OSI a TCP/IP model
#    - IP adresace, subnetting, VLSM
#    - Směrování (statické, dynamické - OSPF, RIP)
#    - DNS, DHCP, NAT, VPN

# 2. OPERAČNÍ SYSTÉMY (Linux)
#    - Instalace a konfigurace
#    - Správa uživatelů, práv, služeb
#    - Bash skriptování
#    - Apache, Nginx, PHP, MySQL

# 3. BEZPEČNOST
#    - Firewally (iptables, nftables)
#    - SSH, SSL/TLS, certifikáty
#    - Obrana před útoky (DDoS, MITM)

# 4. VIRTUALIZACE A CLOUD
#    - Docker, Kubernetes
#    - Výpočetní modely (IaaS, PaaS, SaaS)
#    - VMware, Proxmox

# 5. MONITORING A SPRÁVA
#    - SNMP, Zabbix, Nagios
#    - Logování, analýza provozu

3 — Příklady maturitních otázek

Ukázkové otázky 
# Praktické zadání (příklad):
# "Nakonfigurujte na routeru statické trasy tak, aby..."
# "Vytvořte bash skript, který bude každou hodinu..."
# "Nainstalujte a nakonfigurujte LAMP stack..."
# "Vytvořte Dockerfile pro Python aplikaci..."

# Teoretické otázky:
# - Vysvětlete rozdíl mezi TCP a UDP
# - Jak funguje ARP protokol?
# - Popište proces zálohování a obnovy MySQL
# - Jaká je role reverse proxy (Nginx)?
# - K čemu slouží systemd a jaké má jednotky?
